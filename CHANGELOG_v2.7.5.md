# v2.7.5 更新日志

## ⚡ WebDAV 检测性能优化

### 优化内网检测速度
- **问题**：内网检测需要等待4秒，效率太低
- **原因**：串行检测（逐个尝试），超时设置不合理
- **优化**：
  - 改为并发检测：同时检测所有候选端点，谁先成功用谁
  - 优化超时设置：
    * 内网检测：4秒 → **800ms**（快5倍）⚡
    * 外网检测：4秒 → **3秒**
    * 不匹配检测：2秒 → **1.5秒**
  - 拆分检测逻辑为独立方法，支持并发调用
- **效果**：
  - 内网环境：检测速度提升 **4-8倍**（4秒+ → <1秒）🚀
  - 外网环境：检测速度提升 **1.5-2倍**（3秒+ → <2秒）
  - 配置错误：检测速度提升 **2.6倍**（8秒 → 3秒）

## 技术改进

- 使用 `Future.any()` 实现并发竞争检测
- 使用 `Future.wait(eagerError: false)` 收集完整错误信息
- 保持缓存机制不变（45秒/5秒）
- 网络环境判断错误时也不影响（并发都尝试）

## 性能提升对比

| 场景 | 优化前 | 优化后 | 提升 |
|------|--------|--------|------|
| 内网正常 | 4秒+ | <1秒 | **5倍+** ⚡ |
| 外网正常 | 3秒+ | <2秒 | **2倍** ⚡ |
| 配置错误 | 8秒 | 3秒 | **2.6倍** ⚡ |
