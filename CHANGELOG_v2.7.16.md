# v2.7.16 更新日志

## 拼音小耳朵修正

### 声母不再带声调
汉语拼音声母本身不标调，声调只出现在韵母和整体认读音节上。

**修复前**
- 声母分区也会显示“一声 / 二声 / 三声 / 四声”
- 大卡片会把声调符号标到声母上，容易误导孩子

**修复后**
- 声母分区隐藏四声选择
- 声母标题只显示无调字母（如 `b`、`zh`、`f`）
- 声母固定使用教学读音播放，并提示“声母本身没有声调”
- 韵母、整体认读仍可练习四声

### 发音方法改为中文呼读
大卡片“发音方法”播报原先直接把 `f`、`bo`、`fo` 等拉丁字母交给 TTS，中文语音常会按英文读。

典型问题：
- 声母 `f` 被读成英文 **ef**，听起来像“e”
- 说明文字里的 `d` / `g` / `j`、示例音节 `fo` / `te` 等也容易读错

**修复后**
- 界面继续显示拼音字母，方便认读
- TTS 使用教学呼读中文播报（如 `f` → 佛，`b` → 玻）
- 说明与例句中的拼音也会转成中文读法后再播报

### 文案微调
- 统一部分送气对比说明（`t` / `k` / `q`）
- 弱化技术向提示，保留孩子可理解的发音引导
